Opdracht 3 - Weegbrug
Analyse
Probleemstelling:
De objectgeoriënteerde begrippen implementeren in de gekozen objectgeoriënteerde programmeertaal, voor een industriële omgeving.
Gegeven: 
Een weegbrug met:
afgeschermde eigenschappen: gewichtLinks, gewichtMidden, gewichtRechts
niet-afgeschermd gedrag: getGewicht ==> berekent het totaalgewicht (= som van 3 weegsensoren)
Een lichtkrant met:
afgeschermde eigenschappen: boodschap, actief
niet-afgeschermd gedrag: toonBoodschap ==> toon de boodschap in het console venster
Een vrachtwagen met:
niet-afgeschermde eigenschappen: klantNaam
Een grondstof met:
niet-afgeschermde eigenschappen: prijsPerKilogram
Gevraagd: Realiseer de broncode voor
de klassen Weegbrug; Lichtkrant; Vrachtwagen; Grondstof
een hoofdalgoritme WeegToepassing met:
instanties truckJan; opleggerMieke, grondstofZand, grondstofSteen, beeldscherm, weegschaal
Algoritme
Het programma laat Jan zand kopen en Mieke stenen kopen. De klantnaam en prijs wordt getoond door de lichtkrant.

Coderen
Het hoofdalgoritme en elke klasse in de gekozen objectgeoriënteerde programmeertaal.
